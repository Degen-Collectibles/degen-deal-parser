Degen Collectibles QuickBooks Security Overview

Effective date: May 14, 2026

Scope

This overview describes the security controls for the Degen Collectibles operations portal and its QuickBooks integration. The integration is for internal bookkeeping and reconciliation workflows only.

Access Control

  • Application access is role-based and limited to authorized staff.
  • Finance and bookkeeping pages require authenticated administrative access.
  • Access is removed when a user no longer needs it for business duties.

OAuth and Token Handling

  • QuickBooks authorization uses Intuit OAuth 2.0.
  • Client secrets and runtime credentials are stored outside source code.
  • Refresh tokens are used only to maintain authorized sync access.
  • Tokens are revoked or rotated if access is no longer needed or compromise is suspected.

Data Protection

  • Production access is served over HTTPS.
  • QuickBooks data is used only for internal accounting and reconciliation.
  • QuickBooks data is not sold or exposed to unauthorized third parties.
  • Sensitive operational data is restricted to personnel with a business need.

Logging and Monitoring

Sync failures, import status, reconciliation mismatches, parsing errors, and operational exceptions are visible to authorized staff through application pages or logs. The application avoids logging passwords, OAuth secrets, or full sensitive token values.

Change Management

Application changes are made in the source repository, reviewed for scope, tested where practical, and deployed through the approved hosting workflow. Changes involving authentication, webhooks, financial calculations, token handling, or reconciliation logic receive additional care before production use.

Incident Response

If a security incident is suspected, Degen Collectibles will contain access, preserve relevant logs, rotate affected credentials, revoke affected OAuth connections where needed, assess affected data, document corrective action, and notify affected parties, vendors, or regulators when legally required.

Contact
