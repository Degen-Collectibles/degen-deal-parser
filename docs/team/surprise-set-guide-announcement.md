# Announcement Draft: Surprise Set Guide

Title: New guide: running TikTok Surprise Sets

Body:

We added a streamer guide for running official TikTok Surprise Sets.

Read it here: /team/documents

Use it before running a Surprise Set stream. The guide covers how to build the pool, what products usually go into these sets, how to set up the camera/product wall, how to explain the bottom TikTok panel, and what to say during the auction.

The main rhythm:

- Keep the hit wall on screen.
- Tell viewers to tap the Surprise Set card.
- Tell viewers to tap their own Surprise Set card and check available/sold.
- Say the remaining count and top hits.
- Keep the auction moving.
- Use pressure lines like "insane odds right now" or "these bids are light" when the visible set panel supports it.
- Do not promise guaranteed value, profit, or "you cannot lose."

Managers: make sure every streamer reads this before running Surprise Sets.
